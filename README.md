📱 Aplicativo de Vacinação Infantil

Este projeto foi desenvolvido como parte de um desafio técnico utilizando **Angular + Ionic**.
A aplicação tem como objetivo simular o acompanhamento de crianças e suas vacinas, permitindo visualizar informações de forma simples e organizada.

🎯 Objetivo
O sistema foi criado para ajudar no acompanhamento de:
- Crianças cadastradas
- Situação das vacinas de cada criança
- Campanhas de vacinação
🧩 Funcionalidades:

👶 Lista de crianças
- Exibe nome, idade e status geral de vacinação
- Permite acessar detalhes de cada criança

💉 Detalhes da criança
- Lista de vacinas por criança
- Indicação do status:
  - Em dia
  - Pendente
  - Atrasada

📢 Campanhas de vacinação
- Lista de campanhas ativas e encerradas
- Informações simples sobre cada campanha

🎨 Cores utilizadas (obrigatório no desafio)

- Verde: `#ABC270`
- Amarelo: `#FEC868`
- Laranja: `#FDA769`
- Marrom: `#473C33`
As cores ajudam a identificar o status das vacinas.


📱 Responsividade
O projeto foi pensado para funcionar em diferentes tamanhos de tela:
- Celular
- Tablet
- Desktop

🛠️ Tecnologias usadas
- Angular
- Ionic
- TypeScript
- HTML
- SCSS
